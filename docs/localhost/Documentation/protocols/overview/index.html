Overview

Whenever it comes to any form to interprocess communication (be it on the same machine or over any form of network) the base system uses some custom protocols for this documentation.

They are language independent and can be implemented in any kind of programming language capable of socket communication and some kind of data types (they may be tricky to be implemented in brainfuck or other esoteric languages).

In this document, all used custom protocols are explained in detail containing their purpose, structure and other specifications.
